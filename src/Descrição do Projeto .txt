Projeto de Locadora de Veículos

### Entidades Principais:

1. **`Veiculo` (Classe Pai):**
   - A classe `Veiculo` representa os veículos disponíveis na locadora.
   - Atributos podem incluir modelo, ano, cor, placa, preço de locação por dia, status (disponível/locado), etc.

2. **Classes Filho de `Veiculo`:**
   - Pode haver classes específicas para tipos de veículos como Carro, Moto, Caminhão, etc.
   - Cada classe filho herda de `Veiculo` e pode ter atributos específicos.

3. **`Cliente`:**
   - Representa os clientes que podem alugar veículos.
   - Atributos podem incluir nome, CNH (Carteira Nacional de Habilitação), histórico de locações, etc.

4. **`Locacao`:**
   - Representa uma transação de locação, associando um cliente a um veículo.
   - Atributos podem incluir data de início, data de término, valor total, status (ativa/encerrada), etc.
   - Métodos podem incluir cálculos de custo com base nas taxas diárias do veículo.

### Funcionalidades e Métodos:

1. **Cadastro de Veículos e Clientes:**
   - Métodos para adicionar, remover e atualizar informações de veículos e clientes na locadora.

2. **Locação e Devolução:**
   - Métodos para realizar operações de locação e devolução de veículos.
   - Verificação de disponibilidade do veículo antes de uma locação.

3. **Relatórios e Estatísticas:**
   - Métodos para gerar relatórios, como histórico de locações, veículos mais alugados, etc.

4. **Cálculo de Custos:**
   - Métodos para calcular os custos de locação com base nas taxas diárias, duração da locação, etc.

5. **Gerenciamento de Status:**
   - Métodos para atualizar o status dos veículos (disponível/locado) e das locações (ativa/encerrada).

### Estrutura de Dados:

1. **Listas para Armazenamento:**
   - Utilização de listas para armazenar informações sobre veículos, clientes e transações de locação.

2. **Herança e Polimorfismo:**
   - Hierarquia de classes que permite a extensão para diferentes tipos de veículos.
   - Polimorfismo para operações comuns, como cálculos de custo.

### Exemplo de Uso:

```java
LocadoraVeiculos locadora = new LocadoraVeiculos();

// Adicionar veículos e clientes
locadora.cadastrarVeiculo(new Carro("Modelo", "ABC123", 2022, "Vermelho", 100.0));
locadora.cadastrarCliente(new Cliente("João", "123456789"));

// Realizar locação
locadora.realizarLocacao("ABC123", "123456789", "2023-11-01", "2023-11-05");

// Gerar relatório
locadora.gerarRelatorioHistoricoLocacoes();
```
